## How to use this app locally?

**Pre-requisites**

- Have Node installed on your machine.

### Setting up instructions

**In case you use Git**

1. Open terminal

2. Run the following commands:

- git clone https://github.com/syritchenkom/bookmark.git

- cd bookmark

- yarn install

- yarn start
